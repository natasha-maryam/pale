FEATURE 1

Choose available tools

Make tools available to the AI by simply @mentioning them. Convenient defaults: reading allowed, asks before making changes (writes).

How it works:

1

Type @ to see available tools

2

Tool chips appear as you @mention them

3

Each tool has default permission settings

FEATURE 2

Stay in charge

You choose Allow once, Always allow, or Deny for each action. Never worry about AI overstepping boundaries.

Granular Control

Permission requests appear whenever the AI needs to perform a write action.

Permission Memory

"Always allow" remembers your preferences for future interactions.

Transparent Actions

Clear visual cues show exactly what actions the AI is requesting.
